# 🚀 HabitPulse

Track your habits. Build consistency.

---

## 📌 Overview

HabitPulse is a simple habit tracker web app that helps users track daily habits, maintain streaks, and visualize progress over time.

---

## ✨ Features

* Add new habits
* Mark habits as Done or Missed
* Track streaks, completed, and missed counts
* Data persistence using localStorage
* Clean and responsive UI
* (In progress) Yearly tracking with heatmap
